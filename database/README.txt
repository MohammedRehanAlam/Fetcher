FETCHER — DATABASE FOLDER
=========================

Place your documents here to use the "Pre-loaded Database" mode.

1. SUPPORTED FORMATS:
   - PDF, DOCX (Word), PPTX (PowerPoint), XLSX (Excel)
   - TXT, CSV, RTF
   - ODT, ODP, ODS (OpenOffice)

2. ORGANIZATION:
   - You can create any number of sub-folders (e.g. State -> District -> Constituency).
   - Fetcher will automatically build a dropdown menu for these folders.

3. IMPORTANT:
   - Every time you add or remove files in this folder, you MUST run
     the 'generate-manifest.bat' file located in the main project folder.
   - This updates 'manifest.json' so the website knows which files to search.

4. USAGE:
   - Open index.html and click "Use Database Folder".
